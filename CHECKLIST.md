# find_jboss — Checklist

Purpose: Utility to locate JBoss-related files and roles via Ansible playbooks and scripts.

Prerequisites
- `ansible-core` (for playbooks)

Quick start
```bash
cd find_jboss
ansible-playbook playbooks/find_jboss.yml
```

Verify
- Review the output of the playbook and `reports/` for matches

Notes
- See `README.md` for specific playbook names and variable usage.
